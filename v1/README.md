# Abhyasa v1

**Custody Transfer of Governance Obligations over Unreliable Agent Channels**

This URI (`https://ravikiran438.github.io/abhyasa-protocol/v1`) identifies
version 1 of the Abhyasa protocol extension for A2A and MCP. An agent advertises
support by an entry in `AgentCard.capabilities.extensions[]` whose `uri` equals
this value; the entry body deserializes to `AbhyasaServiceRef`
(`manifest.json` publishes the exact schema).

- **Specification:** See `SPEC.md` and the repository root
- **Status:** Draft v0.1.0
